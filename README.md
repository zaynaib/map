
# Neighborhood Map

This is a neighborhood map the displays the museums in Chicago.

## Technologies 

- React
- FourSquare
- Google API

## Installation



## How to Run App

In order to install the necessary packages to run
this app in the command line `npm i`

To run the app in the command line type `npm start`
